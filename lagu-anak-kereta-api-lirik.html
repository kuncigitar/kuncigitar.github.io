9. Lagu naik kereta api dinyanyikan dengan tempo?


Lagu Naik Kereta Api dinyanyikan dengan tempo SEDANG atau moderato. Lagu Naik Kereta Api adalah salah satu lagu anak Indonesia yang paling terkenal.

» Pembahasan

Lagu Naik Kereta Api adalah ciptaan Ibu Sud. Ibu Sud bernama lengkap Saridjah Niung. Beliau adalah musisi kawakan Indonesia yang dikenal sebagai pencipta lagu anak legendaris. Ia lahir pada tahun 1908 dan wafat pada tahun 1993. Ada puluhan lagu anak yang sudah diciptakan oleh Ibu Sud dan banyak di antaranya yang terkenal dan terus dinyanyikan anak-anak sampai saat ini.  

Selain Naik Kereta Api, lagu ciptaan Ibu Sud yang juga terkenal adalah Burung Kutilang, Naik-naik ke Puncak Gunung, Naik Delman, Tanah Airku, Berkibarlah Benderaku, Hai Becak dan lain-lain.

30. Sebuah kereta api meluncur menjauhi stasiun kereta api dengan kecepatan 72 km/jam. Seorang anak berjalan searah kereta api dengan kecepatan tetap 5 m/s. Berapakah kecepatan anak terhadap stasiun kereta api ?


Penjelasan:

Sebuah kereta api meluncur menjauhi stasiun kereta api dengan kecepatan 72 km/jam. Seorang anak berjalan searah kereta api dengan kecepatan tetap 5 m/s. Berapakah kecepatan anak terhadap stasiun kereta api ?

Diketahui,

V1 (Kereta api) = 72 km/jam

V2 (seorang anak) = 5 m/s

Untuk mencari yang lebih cepat, kita samakan satuan kecepatannya.

72 km/jam

[tex] \frac{72000 \: m}{3600 \: s} = \frac{720 \: m}{36 \: s} = 20 \frac{m}{s} [/tex]

72 km/jam = 20 m/s

Dibanding dengan seorang anak itu, 5 m/s selisih 15 m/s dengan kecepatan kereta api.

Kecepatan anak tersebut, kurang 15 m/s dari kecepatan kereta api.
